# 酷家乐图纸脚本

## 核心流程

1. 明确图纸类型、入口函数和输出对象。
2. 统一坐标系、单位、旋转和偏移定义。
3. 区分模型轮廓、台面轮廓、孔洞和挡水数据。
4. 用最小几何场景验证，再覆盖 L/U 型等复杂场景。
5. 检查空数据、共线点、闭合轮廓和重复尺寸。
6. 使用真实图纸输出进行视觉和数据验证。

## 质量要求

- 记录坐标系和单位。
- 提供简单、转角和异常轮廓测试。
- 不把设计模型尺寸直接等同于加工尺寸。
