# 安全与脱敏规范

本仓库按公开仓库标准维护。

## 禁止提交

- 密码、Token、Cookie、Access Key、内部认证配置；
- 客户姓名、电话、地址、订单号、设计方案原始 JSON；
- 未脱敏日志中的用户 ID、hunterId、taskId、账号 ID；
- 企业专属商品 ID、价格、供应链映射，除非已获得公开授权；
- 内部源代码、未公开接口实现或受限文档全文；
- 未经授权的内部 CF 截图和附件。

## 可以提交

- 已脱敏的脚本结构和通用算法；
- 使用占位符的商品映射；
- 内部文档标题和结论摘要；
- 不包含身份信息的最小日志片段；
- 经授权可公开的链接和资料。

## 占位符示例

```text
BG_ID_FOR_20MM
BG_ID_FOR_12MM
TASK_ID_REDACTED
USER_ID_REDACTED
```

发现敏感信息时不要只在新提交中删除。立即停止推送，并联系仓库管理员清理 Git 历史和轮换相关凭证。
